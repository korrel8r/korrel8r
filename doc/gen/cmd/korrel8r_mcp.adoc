= korrel8r mcp

MCP stdio server

== Synopsis

Run korrel8r as an MCP server communicating via stdin/stdout.
Allows korrel8r to be run as a sub-process by an MCP tool.
For a HTTP streaming server use the 'web' command with the '--mcp' flag.

----
korrel8r mcp [flags]
----
