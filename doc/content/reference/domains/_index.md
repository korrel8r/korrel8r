# Domains

A _domain_ represents one type of signal or resource.

This reference defines the following for each domain:

* Object: Individual data items returned by queries,
* Store: Connection URL to backend service that holds the data.
* Class: The class or classes defined by the domain.
* Query: A request for data, formatted as `domain:class:selector`.
